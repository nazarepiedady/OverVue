# Control Structures

## Conditionals

### if ... else

```js {2-6}
let player = 'Michael Jordan'
if (player === 'Michael Jordan') {
  console.log('Yay, I love Michael Jordan!')
} else {
  console.log('Awwww, no...')
}
```

### if ... else if ... else

```js {2-8}
let score = 0
if (score < 3) {
  console.log('Score more')
} else if (score >= 10) {
  console.log('Go on')
} else {
  console.log('Good Job')
}
```
